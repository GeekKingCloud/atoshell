# Security

Atoshell is a local CLI that reads and writes project-local `.atoshell/` JSON files.

Please do not store secrets, credentials, API keys, tokens, or private keys in ticket titles, descriptions, comments, or config values.

If you find a bug that could cause unsafe shell execution, writes outside `.atoshell/`, unsafe install/update behavior, terminal escape injection, or exposure of local data, please use GitHub's private vulnerability reporting for this repository if it is available.

If private reporting is not available, open a regular issue with only a short summary and ask for a private contact before sharing exploit details, local paths, secrets, logs, or proof-of-concept commands.

For ordinary crashes, bad output, validation errors, or workflow bugs, open a regular issue.
